# Command Line Portfolio

This is a CLI-style portfolio website built with React, TypeScript, and Styled-Components.
It includes multiple themes, keyboard shortcuts, and command-line features.

This project is a fork of the Terminal Portfolio by Sat Naing, with custom modifications and improvements.

## Running Locally

Clone the project

```bash
git clone https://github.com/krdgomer/cli-portfolio.git
```

Go to the project directory

```bash
cd cli-portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
